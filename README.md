To run the application, click on https://lachezartsk.github.io/Invoice-Generator/

Information about the application can be found in file "Project Summary". It is 14 pages and if it is viewed on GitHub, the complete file is not loaded in one piece but in several stages. So, after scrolling for a while, if a button "More Pages" appears at the bottom, click on it to load the next few pages of the file. As long as this button appears after scrolling a few pages, the complete file is not viewed. Or alternatively, to avoid this, the file could simply be downloaded.

A sample invoice in PDF format is in file "Sample Invoice in PDF format". It is 2 pages and the complete file can be viewed as one piece on GitHub.
